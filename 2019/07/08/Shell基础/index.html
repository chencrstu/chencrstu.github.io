Shell基础

最近在折腾flutter，iOS及Android工程接入Flutter进行混合编译基本搞定。
构建出包及静态检测存在一些不同，这里简单梳理一下Shell编程的基础部分，提高编写的脚本质量。

变量

shell中变量分为两种，普通变量，环境变量。

环境变量亦称为全局变量，可以在建立它们的shell及其派生出来的任意子进程shell中使用。Shell通过环境变量来确定登陆用户名，命令路径，终端类型，登录目的。

普通变量亦称为局部变量，只能在创建它们的shell函数或shell脚本中使用。

运算符

原生shell不支持数学运算，需要借助awk或者expr实现
